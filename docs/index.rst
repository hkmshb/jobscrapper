Jobs Scraper
============

A web scraper for extracting job/opening listings from the `World Bank Group <https://worldbankgroup.csod.com/ats/careersite/search.aspx>`_
and `SEQUOIA <https://www.sequoiacap.com/jobs>`_ websites.
